Getting Started

Welcome to the Catering Management app! This tool is designed to be your all-in-one solution for managing clients, creating a library of your dishes, and planning events down to the last detail.

Core Concepts

The app is built around three main components: your Dish Library, your Clients, and your Events. For the best experience, it's recommended to add some dishes and clients before creating your first event.

  • Data Storage: All your data is saved instantly and securely in your browser's local storage. There's no need to manually save your work.
  • Backups: For safety and portability, you can back up your data to a file on your computer using the features on the Preferences page.

Feature Guide

Dashboard

The dashboard is your home screen. It provides an at-a-glance view of all events scheduled within the next 7 days, helping you focus on what's immediately important.

Dish Library

This is where you build your menu. You can add, edit, and delete every dish your company offers. Each dish includes a name, category, price per serving, and a description. This library is used to build menus when you create an event.

Client Management

Keep track of your customers here. You can add new clients or edit existing ones. When you create an event, you will select a client from this list.

Event Management

The Event List

This page lists all your events. It's designed for quick access and powerful filtering:

  • Sorting: Events are sorted to show upcoming events first (starting with the soonest) and past events last.
  • Highlighting: Events happening today are highlighted with a light blue background and a side border for easy identification.
  • Search: Use the search bar to instantly find events by a client's name or the event's venue.
  • Filters: You can filter the list to show only events with a specific status (e.g., "Confirmed") or view all events scheduled for a specific date using the date picker.

Creating & Editing an Event

This is the most detailed page. When you create or edit an event, you can:

  • Select a Client: Start typing a client's name in the searchable dropdown to quickly find and select them.
  • Set Event Details: Add the date, time, venue, and guest count. The app will warn you if you select a date where the client already has another booking. You cannot create new events in the past.
  • Build a Menu:
    • Click "Add Dish" to open a pop-up of your entire Dish Library.
    • As you select dishes, the pop-up shows you an estimated cost based on the guest count.
    • When you add dishes, their quantity will automatically default to the event's guest count. You can then change the quantity for each dish individually.
    • If you change the main guest count, the app will ask if you want to update all dish quantities to match.
    • The Total Estimated Cost updates in real-time as you build the menu.

Preferences & Data Management

This page gives you full control over your data backups.

Auto-Save (Recommended)

This is the best way to keep your data safe. After a one-time setup where you choose a file on your computer, the app will automatically save a backup to that file every 15 seconds, but only if you've made changes. This is highly efficient and provides peace of mind against data loss.

Manual Import / Export

You can also manually save a complete snapshot of your data (Export) or restore your app from a backup file (Import). This is useful for moving your data to a new computer.

Frequently Asked Questions (FAQ)

Can I use this app on multiple devices?

Not simultaneously. The data is stored locally in the browser on a single computer. To move your data from one computer to another, use the Export feature on the old computer and the Import feature on the new one.

What happens if I delete a Client or a Dish?

If you delete a client or a dish, any events that used them will be affected. A deleted client's name will show as "N/A" in the event list, and a deleted dish will disappear from any event menus it was part of. This action cannot be undone, so please be careful.
